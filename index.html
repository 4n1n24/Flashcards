Qual a definição de Cinematografia?

o conjunto de princípios, processos e técnicas utilizados para captar e projetar numa tela imagens estáticas sequenciais (fotogramas) obtidas com uma câmera especial, dando impressão ao espectador de estarem em movimento.

O que é audiovisual?

Audiovisual é uma forma de comunicação que combina elementos visuais e sonoros, podendo ser vista e ouvida ao mesmo tempo. É uma linguagem de síntese que reúne imagens, sons, cores, palavra escrita e movimento para expressar ideias, pensamentos e juízos.

Por que o cinema é considerado uma arte audiovisual?

O cinema é considerado uma arte audiovisual porque combina elementos visuais e sonoros para contar uma história ou transmitir uma mensagem.
